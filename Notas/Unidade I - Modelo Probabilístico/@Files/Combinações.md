Queremos contar o número de subconjuntos de $r$ elementos a partir de um conjunto de $n$ elementos ($r < n$). 
- A ordem não é importante. 

O número total de resultados possíveis é dado por:

$C^n_r =  \binom{n}{r}  = n!/ (n − r)!r!$

- Exemplo:
Um componente pode ser colocado em 8 localizações diferentes uma placa de circuito. Cinco componentes idênticos são colocados na placa. Cada projeto é um subconjunto das 8 localizações que podem conter os componentes.

O número total de projetos é $C^8_5 = 8!/5!3!$
